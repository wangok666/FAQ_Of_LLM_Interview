## 针对信息物理系统（Cyber-Physical Systems, CPS）中线性欺诈攻击的水印加密策略

通过在系统中注入一种秘密的、随机的“水印”信号，并结合特定的检测机制（如KL散度检测器）

1.  **问题背景**：在信息物理系统中，传感器采集的数据通常通过无线网络传输至远程估计器。攻击者可以截获并篡改这些测量数据，实施线性欺诈攻击。这种攻击旨在降低远程状态估计的性能，同时尽可能地规避传统的异常检测器（如卡方检测器），使其难以被发现

2.  **水印加密策略**：为了应对上述挑战，该策略在系统的控制输入或传感器端引入一个秘密的、随机的水印信号。这个水印是系统内部已知但对外部（包括潜在攻击者）保密的。它的主要作用是为正常的系统行为“签名”，任何对测量数据的篡改都会破坏这种由水印建立的独特统计特性 

3.  **检测机制（KL散度检测器）**：系统采用基于Kullback-Leibler (KL) 散度的检测器来监控接收数据的统计特性。KL散度是一种衡量两个概率分布差异的指标。检测器会将接收到的数据与预期的、包含合法水印信号的数据的概率分布进行比较。如果攻击者篡改了数据，其统计特性将偏离预期分布，导致KL散度显著增大，从而触发警报

4.  **防御效果**：通过这种方式，即使攻击者能够构造出能骗过传统检测器的线性攻击，也很难在不知道秘密水印的情况下，同时伪造出符合正确统计特性的虚假数据。因此，该策略有效地提高了系统对线性欺诈攻击的检测能力和整体安全性

```python
import numpy as np
import matplotlib.pyplot as plt

# 设置中文字体和解决负号显示问题
plt.rcParams["font.sans-serif"] = ["SimHei"]  # 用黑体显示中文
plt.rcParams["axes.unicode_minus"] = False  # 正常显示负号

# 设置随机种子（便于复现）
np.random.seed(42)

# 系统参数（一维线性系统）
A = 1.0  # 状态转移矩阵（标量）
C = 1.0  # 观测矩阵
Q = 0.1  # 过程噪声方差
R = 0.1  # 测量噪声方差
watermark_power = 0.5  # 水印强度（方差）

T = 100  # 时间步长

# 1. 生成真实系统状态和无水印测量
x_true = np.zeros(T)
y_clean = np.zeros(T)
for t in range(1, T):
    x_true[t] = A * x_true[t - 1] + np.random.normal(0, np.sqrt(Q))
    y_clean[t] = C * x_true[t] + np.random.normal(0, np.sqrt(R))

# 2. 传感器端注入秘密水印（仅合法系统知道）
w = np.random.normal(0, np.sqrt(watermark_power), T)  # 秘密水印
y_watermarked = y_clean + w  # 合法传输的数据

# 3. 模拟攻击者行为（线性欺诈攻击，但不知道水印）
# 攻击者试图发送 y_attack = y_clean + a * y_clean （或其他线性变换）
a = 0.2  # 攻击增益
y_attacked = y_clean + a * y_clean  # 注意：攻击者未加水印！

# 4. 接收端检测：比较接收数据与预期水印统计特性
# 合法数据应满足：y ~ N(μ, R + watermark_power)
# 我们用样本方差作为统计特征（简化KL检测）

var_legitimate = np.var(y_watermarked)
var_received_normal = np.var(y_watermarked)  # 无攻击
var_received_attacked = np.var(y_attacked)  # 有攻击

print(f"合法数据方差: {var_legitimate:.3f}")
print(f"攻击数据方差: {var_received_attacked:.3f}")

# 简单阈值检测（实际中可用KL散度）
threshold = var_legitimate * 0.9  # 容忍一定波动
if var_received_attacked < threshold or var_received_attacked > var_legitimate * 1.2:
    print("⚠️ 检测到异常：可能存在欺诈攻击！")
else:
    print("✅ 数据正常")

# 可视化（可选）
plt.figure(figsize=(10, 4))
plt.plot(y_watermarked, label="合法数据（含水印）", alpha=0.8)
plt.plot(y_attacked, label="攻击数据（无水印）", linestyle="--")
plt.legend()
plt.title("Watermark-based Attack Detection (Simplified)")
plt.xlabel("Time step")
plt.ylabel("Measurement")
plt.grid(True)
plt.show()
```
